# 🛍 ShopNest - E-commerce Website

ShopNest is a simple and responsive **E-commerce Website** built using **HTML, CSS, and JavaScript**.  
It includes product cards, add to cart functionality, quantity update, cart sidebar, and total price calculation.

## 🚀 Features

- 🏠 Attractive Homepage
- 🛍 Product Listing Section
- 🛒 Add to Cart Functionality
- ➕ Increase / Decrease Quantity
- ❌ Remove Items from Cart
- 💰 Total Price Calculation
- 📱 Responsive Design
- 🎨 Clean and Modern UI
- ⚡ Beginner Friendly Code Structure
